## Points Positifs

### Backend
- Requetes SQL parametrees (pas d'injection SQL)
- Auth JWT sur toutes les routes
- Pool DB singleton
- Whitelist des colonnes pour le tri
- Validation UUID + pagination
- Client Sophia GO reutilisable avec refresh token

### Frontend
- SWR pour le data fetching (cache, revalidation)
- DataTable generique reutilisable
- Animations soignees (anime.js stagger, spring hover)
- Gestion des etats loading/empty
- Systeme de 3 regles de validation visuel (dots)
- Panneau flottant draggable
